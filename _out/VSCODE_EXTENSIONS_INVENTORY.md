## 🔄 問題排查流程

### 🚨 常見問題與解決方案

| 問題類型 | 症狀 | 解決步驟 |
|----------|------|----------|
| **插件衝突** | 功能異常、效能下降 | 1. 停用近期安裝插件<br>2. 逐一測試啟用<br>3. 查看擴充功能日誌 |
| **Python 環境** | Pylance 無法運作 | 1. 檢查 Python 解釋器<br>2. 重新載入視窗<br>3. 清除 Pylance 快取 |
| **Git 功能異常** | GitLens 無顯示 | 1. 確認 Git 安裝<br>2. 檢查倉庫狀態<br>3. 重新整理 Git 狀態 |
| **效能問題** | VS Code 緩慢 | 1. 停用不必要插件<br>2. 增加記憶體配置<br>3. 清除擴充功能資料 |

### 🔧 維護建議

| 維護項目 | 頻率 | 操作步驟 |
|----------|------|----------|
| **插件更新** | 每週 | 自動更新或手動檢查更新 |
| **設定備份** | 每月 | 使用 Settings Sync 備份配置 |
| **效能檢查** | 每季 | 檢查插件使用率，清理不需要的插件 |
| **安全掃描** | 每季 | 執行 Snyk 安全掃描，更新依賴 |
