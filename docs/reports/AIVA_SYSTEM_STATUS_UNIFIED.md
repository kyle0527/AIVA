## 🏗️ 系統架構概況

### 核心模組架構
AIVA 採用五大核心模組架構：

```
services/
├── core/aiva_core/           # 🧠 核心引擎
│   ├── ai_engine/           # BioNeuron AI智能體
│   ├── attack/              # 攻擊執行和漏洞利用 (2025.10.23重組)
│   ├── state/               # 系統狀態管理  
│   └── output/              # 輸出處理
├── scan/                    # 🔍 掃描引擎
│   ├── aiva_scan/          # 主掃描器
│   └── unified_scan_engine.py # 統一掃描接口
├── integration/             # 🔗 整合服務
│   └── aiva_integration/    # 數據整合和經驗學習
├── features/                # 🛠️ 功能檢測
│   ├── function_sqli/       # SQL注入檢測
│   ├── function_xss/        # XSS檢測
│   └── [其他專項功能...]
└── aiva_common/             # 📚 共享基礎設施
```
